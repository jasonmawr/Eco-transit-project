# Content, Review and Guide Workflow

## Content types

- Article.
- Video guide.
- Station guide.
- Place guide.
- User review.
- Admin announcement.

## Public content rules

1. Admin-authored content may be draft/published/archived.
2. User-generated content always starts as pending.
3. Rejected content must store a reason visible to author in friendly Vietnamese.
4. Approved UGC can be edited by author only by creating a new pending revision.
5. Media must go through file validation and private-to-public publishing flow.
6. Offensive/spam/unsafe content must not be published.

## Content metadata

- station_id optional.
- place_ref optional.
- weather_context tags: rain, heat, sunny, night, weekend.
- transit_mode tags: metro, bus, walking, mixed.
- difficulty: beginner, normal, advanced.
- language.
- status.
- published_at.

## Acceptance

- Admin can create a video guide linking a station.
- User can submit review for a place near a station.
- Moderator can approve/reject with reason.
- Public station page shows approved reviews/guides only.
