| Domain | P0 production-grade MVP | P1 near-term | P2 later |
|---|---|---|---|
| Routing | Route search, map, weather-aware scoring, route snapshots | Saved routes, more filters | Predictive crowd/ETA, mood-based routing |
| Stations | Seeded station/stop pages and nearby content | Rich content editor | Community station hubs |
| Ticket | Upload, secure storage, OCR/rule/manual review pipeline | Better duplicate/fraud heuristics | Advanced anti-fraud and partner validation |
| Points | Ledger, balance cache, quotas | Campaign bonuses | Group/campus leaderboards |
| Vouchers | Manual code/CSV/import style | Expiring reminders, batch import | Partner API realtime redemption |
| Bill | Share URL/text, copy fallback, privacy safe | OG image/card templates | Animated cards and personalization |
| UGC | Text/image review with moderation | Rich guide/video | Creator hub |
| Notification | Optional in-app status basics | FCM push | Contextual habit reminders |
| Admin | Moderation queues and voucher management | Content/geofence admin | Analytics dashboard |
| UX | Mobile-first polished screens | More motion/brand assets | Advanced interactive storytelling |
