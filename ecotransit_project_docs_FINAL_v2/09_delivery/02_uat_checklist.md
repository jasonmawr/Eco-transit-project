# UAT Checklist

## Guest

- [ ] Can open app on mobile.
- [ ] Can search route.
- [ ] Can understand weather recommendation.
- [ ] Can view station content.
- [ ] Can complete quiz.
- [ ] Can view public bill link.

## User

- [ ] Can register/login.
- [ ] Can upload ticket.
- [ ] Can see verification status.
- [ ] Can see points ledger.
- [ ] Can redeem voucher.
- [ ] Can create/share bill.
- [ ] Can submit review.

## Moderator/Admin

- [ ] Can view ticket queue.
- [ ] Can approve/reject ticket.
- [ ] Can view review queue.
- [ ] Can approve/reject review.
- [ ] Can manage voucher.
- [ ] Can view audit trail.

## Cross-cutting

- [ ] Friendly Vietnamese messages.
- [ ] No raw provider/DB errors.
- [ ] Mobile 390px no overflow.
- [ ] Privacy text clear.
- [ ] Loading/empty/error states present.
