# Vietnamese Copywriting Rules

## Voice

Friendly, clear, confident, youthful but not childish.

## Rules

- Use Vietnamese labels first.
- Avoid raw technical strings such as `manual_review`, `station_only`, `route_provider_unavailable`.
- Use “ga”, “trạm”, “tuyến”, “đi bộ”, “đổi tuyến”, “thời gian chờ” consistently.
- Make errors actionable.
- Avoid blame: say “Chưa thể...” instead of “Lỗi...”.

## Examples

| Raw/poor | Approved |
|---|---|
| `manual_review` | Đang chờ kiểm tra thủ công |
| `TICKET_DUPLICATE` | Ảnh vé này có dấu hiệu đã được sử dụng trước đó |
| `No data` | Chưa có dữ liệu phù hợp |
| `null` | Không hiển thị; dùng “Chưa cập nhật” nếu cần |
| `Auth failed` | Đăng nhập chưa thành công, vui lòng kiểm tra lại |
| `Route provider unavailable` | Chưa thể tìm tuyến ngay lúc này, bạn có thể thử lại sau |
