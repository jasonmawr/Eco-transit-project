# Final Risk Register

| Risk | Severity | Mitigation |
|---|---|---|
| Official/public transit APIs incomplete | High | Adapter + seed fallback + honest UI labels |
| Ticket OCR false positive/negative | High | Hybrid verification + manual review + no points until verified |
| Voucher double redemption | High | Transaction + inventory lock + idempotency |
| Privacy leakage in bill | High | Default station_only + coordinate masking |
| Generic UI output from agent | Medium | Taste Skill + screenshot gate |
| Scope creep from vague customer feedback | Medium | Change-control file and signoff script |
| API key/provider outages | Medium | Provider health, fallback, user-friendly partial results |
| Legal/privacy copy incomplete | Medium | Owner legal review before launch |
| Seed data looks empty | Medium | Final seed data plan with demo content |
